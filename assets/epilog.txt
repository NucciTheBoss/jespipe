Command Quick Reference:
--------------------------

- Launch Jespipe:
    mpirun -n $NUM_OF_NODES python main.py /path/to/xml/control/file.xml

- No progress bar run:
    mpirun -n $NUM_OF_NODES python main.py -np /path/to/xml/control/file.xml

- Silent run:
    mpirun -n $NUM_OF_NODES python main.py -s /path/to/xml/control/file.xml
